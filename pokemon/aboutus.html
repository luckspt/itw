About us

This website was developed as a web project for our college course Introduction to Web Technologies.

The project was to create a web browser game where the main theme was the Pokémon universe, in which a player could explore a map and along the way fight battles with a multitude of pokémon species in order to collect them. As our classes progressed throughout the semester, with each new material learned, the website was gradually implemented.
